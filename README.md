Hand Motion Fire Drawing — Quick Interaction Guide 

What you do:

1. Point your index finger at the screen to paint glowing fire particles in mid-air.

2. Move your hand faster to create louder, more energetic sound effects.

3. Spread your hand wide (thumb to pinky) to trigger a bright flash and burst of particles.

Keyboard controls:

SPACE → Toggle drawing on or off
S → Toggle sound
F → Toggle fullscreen
C → Clear all particles
Q → Quit the experience

Tips for best magic:

Stand where your hand is clearly visible to the webcam.
Use smooth sweeping motions for elegant fire trails.
Try quick flicks and wide spreads to create dramatic flashes.
Wave your hand like a tiny comet conductor and let the sparks follow your rhythm.

## Setup & Extraction Guide

### 1. Extract the Project
- If the project is a `.zip` file:
  - **Windows:** Right-click → **Extract All…**
  - **Mac:** Double-click the zip and drag the folder out.
- After extracting, open the project folder containing the Python file.

---

### 2. Open in Terminal
Navigate to the project folder in a terminal:

- **Windows:** Right-click inside the folder → **Open in Terminal / PowerShell**
- **Mac:** Right-click folder → **New Terminal at Folder**

---

### 3. Install Dependencies
Run:

pip install opencv-python mediapipe numpy pygame
or
pip3 install opencv-python mediapipe numpy pygame

then python main.py or python3 main.py
